
# INSTALLATION

## Requirements

- Python 3.11
- pip
- virtualenv
- git

## Installation

1. Clone the repository
2. Create a virtual environment
3. Install the requirements -> `pip install -r requirements.txt`
4. Run the application -> `python main.py`
